Features

More than notifications

Push notification Hook-based

Claude Code done → notification pushed instantly. No polling, no delay.

Two-way interaction Tier 2

Send prompts from your phone, Claude Code receives and processes, response sent back. Approve/deny permissions right in chat.

Auto git diff

Each notification includes file list: added, modified, deleted. Know what Claude Code changed without opening your laptop.

Response summary + cost

Extract response content from JSONL transcript, with processing time.

Self-hosted, code stays on your machine

Whitelist User ID, unauthorized gets silently ignored. Everything runs on your machine.

Setup

3 steps, 2 minutes

Requires Node.js ≥ 18 and a Telegram Bot Token.

1

Create Bot Token

Open @BotFather on Telegram → /newbot → save token.
Get your User ID from @userinfobot.

2

Run one command

$ npx ccpoke

💡 For daily use, install globally: npm i -g ccpoke for faster startup.

3

Done!

Open Telegram → send /start to the bot. Use Claude Code normally → notifications arrive.

Telegram-first — perfect the experience on Telegram first, then expand to other platforms.

Phase 1 · Complete

Notification + Response viewer

Push notification when Claude Code completes, with git diff, token/cost stats, and full response viewer via Telegram Mini App.
